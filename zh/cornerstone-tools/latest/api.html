                                                                                                                                                                                                                              Sets the canvas context transformation matrix so it is scaled to show text more cleanly even if the image is scaled up. See https://github.com/cornerstonejs/cornerstoneTools/wiki/DrawingText (opens new window) for more information

                                                                                                                                                                                                                              Parameters

                                                                                                                                                                                                                              • enabledElement
                                                                                                                                                                                                                              • context
                                                                                                                                                                                                                              • fontSize

                                                                                                                                                                                                                              Returns {fontSize: number (opens new window), lineHeight: number (opens new window), fontScale: number (opens new window)}

                                                                                                                                                                                                                              Trigger a CustomEvent

                                                                                                                                                                                                                              Parameters

                                                                                                                                                                                                                              • el EventTarget The element or EventTarget to trigger the event upon

                                                                                                                                                                                                                              • detail (Object (opens new window) | null) =null The event data to be sent (optional, default null)

                                                                                                                                                                                                                              Returns Boolean (opens new window) The return value is false if at least one event listener called preventDefault().                                                                                                                                                                                                                               # getPlayClipTimeouts

                                                                                                                                                                                                                              [private] Turns a Frame Time Vector (0018,1065) array into a normalized array of timeouts. Each element ... of the resulting array represents the amount of time each frame will remain on the screen.

                                                                                                                                                                                                                              Parameters

                                                                                                                                                                                                                              • vector Array (opens new window) A Frame Time Vector (0018,1065) as specified in section C.7.6.5.1.2 of DICOM standard.
                                                                                                                                                                                                                              • speed Number (opens new window) A speed factor which will be applied to each element of the resulting array.

                                                                                                                                                                                                                              Returns Array (opens new window) An array with timeouts for each animation frame.
