本文档重点介绍Cornerstone Core，它是一个轻量级JavaScript库，用于在支持HTML5画布元素的现代web浏览器中显示医学图像。基石核心并不是一个完整的应用程序本身，而是一个可以作为更大、更复杂应用程序的一部分使用的组件。
